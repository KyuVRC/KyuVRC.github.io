    ║ 4. Soutien mutuel                                                                    ║

    │     of doubt. A strong group is strengthened by mutual support.                      │
    │     Soyons présents les uns pour les autres, que ce soit pour partager des joies,    │

    ║ 5. Équité et impartialité                                                            ║
    ╚══════════════════════════════════════════════════════════════════════════════════════╝
    │   - No one should be favored or excluded. Treat all members of the group equally,    │
    │     without favoritism.                                                              │
    │     Aucune personne ne doit être favorisée ou exclue. Traitez tous les               │
    │     membres du groupe de manière équitable, sans favoritisme.                        │
    │   - In case of conflict between members, try to understand both points of view       │
    │     before taking sides.                                                             │
    │     En cas de conflit entre membres, essayez de comprendre les deux points de        │
    │     vue avant de prendre parti.                                                      │

    ║ 8. Éviter les discussions conflictuelles                                             ║
    ╚══════════════════════════════════════════════════════════════════════════════════════╝
    │   - Sensitive topics (politics, religion, etc.) should be approached with care and   │
    │     always with respect for others' opinions.                                        │
